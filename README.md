# bank-testing

Simple Java bank application without an application framework.

## Requirements

- JDK 21+

## Build

```bash
./gradlew build
```

## Test

```bash
./gradlew test
```

## Run

After you add `com.bulutcan.banktesting.Main`:

```bash
./gradlew run
```

## Project Structure

- `src/main/java`: application and domain code
- `src/test/java`: unit tests
